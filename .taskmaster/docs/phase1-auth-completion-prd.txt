# Phase 1: 핵심 인증 완성 PRD

## 프로젝트 개요
Vue.js 기반 MeongNyang 프론트엔드의 핵심 인증 시스템을 완성하여 사용자 경험을 향상시키고 백엔드 API와의 완벽한 연동을 구현합니다.

## 목표
- OAuth 소셜 로그인 후 계정 연동 확인 화면 구현
- 비밀번호 찾기 및 계정 잠금 해제 기능 구현
- 사용자 프로필 관리 기능 구현
- 백엔드 Java API와의 완벽한 연동

## 기술 스택
- Vue 3 (Composition API)
- Vuetify 3
- Vue Router
- Pinia (상태 관리)
- Axios (HTTP 클라이언트)

## 기능 요구사항

### 1. OAuth 계정 연동 확인 화면
- 기존 이메일 계정과 소셜 계정 연동 여부 확인
- 연동 동의/거부 선택
- 연동 완료 후 자동 로그인
- 연동 거부 시 로그인 화면으로 리다이렉트

### 2. 비밀번호 찾기/계정 잠금 해제
- 이메일을 통한 임시 비밀번호 발급
- 계정 잠금 상태 확인 및 해제
- 보안을 위한 이메일 인증 절차
- 사용자 친화적인 에러 메시지

### 3. 프로필 관리
- 사용자 기본 정보 조회
- 프로필 정보 수정 (이름, 닉네임, 프로필 이미지)
- 비밀번호 변경
- 계정 탈퇴 기능

## 구현 우선순위
1. OAuth 계정 연동 확인 화면 (가장 중요)
2. 비밀번호 찾기/계정 잠금 해제
3. 프로필 관리

## 성공 기준
- 모든 인증 관련 화면이 정상 작동
- 백엔드 API와 완벽한 연동
- 사용자 경험 향상
- 에러 처리 및 유효성 검증 완성
