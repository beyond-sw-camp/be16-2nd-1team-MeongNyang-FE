# 멍냥이 프론트엔드 개발 PRD (Product Requirements Document)

## 프로젝트 개요
멍냥이(MeongNyang)는 반려동물 커뮤니티 및 중고거래 플랫폼입니다. Vue.js 기반의 프론트엔드 애플리케이션을 개발하여 사용자들이 반려동물 관련 일기를 작성하고, 중고거래를 할 수 있는 플랫폼을 제공합니다.

## 기술 스택
- **프레임워크**: Vue.js 3
- **UI 라이브러리**: Vuetify
- **상태 관리**: Vuex/Pinia
- **라우팅**: Vue Router
- **HTTP 클라이언트**: Axios
- **웹소켓**: STOMP/WebSocket
- **테스팅**: Playwright

## 주요 기능 요구사항

### 1. 사용자 인증 및 관리
- 회원가입 (이메일 인증 포함)
- 로그인/로그아웃
- 비밀번호 찾기/변경
- 계정 잠금 해제
- 회원 탈퇴
- 마이페이지 관리
- 팔로우/언팔로우 기능
- 사용자 차단/차단 해제

### 2. 반려동물 관리
- 반려동물 등록/수정/삭제
- 반려동물 목록 조회
- 대표 반려동물 설정
- 반려동물 종류 검색 및 선택

### 3. 일기 작성 및 커뮤니티
- 일기 작성/수정/삭제
- 이미지 업로드 (다중 파일)
- 일기 목록 조회 (내 일기)
- 일기 상세 조회
- 좋아요/좋아요 취소
- 댓글 작성/수정/삭제
- 대댓글 기능
- 일기 검색 기능
- 신고 기능

### 4. 중고거래 마켓
- 거래글 등록/수정/삭제
- 이미지 업로드 (다중 파일)
- 거래글 목록 조회
- 거래글 상세 조회
- 찜하기/찜 취소
- 찜 목록 조회
- 구매/판매 목록 조회
- 카테고리별 분류

### 5. 실시간 채팅
- 채팅방 생성
- 채팅방 목록 조회
- 실시간 메시지 송수신
- 파일 업로드
- 참여자 관리 (초대/나가기)
- 온라인/오프라인 상태 표시
- 메시지 읽음 처리

### 6. 관리자 기능
- 회원 관리
- 신고 처리
- 관리자 권한 확인

### 7. 실시간 알림
- SSE(Server-Sent Events) 연결
- 실시간 알림 수신

## API 엔드포인트 통합

### REST API
- UserRestController: 사용자 관리
- PostRestController: 일기 관리
- MarketController: 중고거래 관리
- ChatController: 채팅방 관리
- PetController: 반려동물 관리
- SpeciesController: 반려동물 종류 관리
- AdminRestController: 관리자 기능
- SseController: 실시간 알림

### WebSocket (STOMP)
- 실시간 채팅 메시지
- 채팅방 참여자 관리
- 온라인/오프라인 상태

## UI/UX 요구사항

### 반응형 디자인
- 모바일, 태블릿, 데스크톱 지원
- Vuetify 컴포넌트 활용

### 사용자 경험
- 직관적인 네비게이션
- 로딩 상태 표시
- 에러 처리 및 사용자 피드백
- 접근성 고려

### 성능 최적화
- 이미지 최적화
- 코드 스플리팅
- 캐싱 전략

## 보안 요구사항
- JWT 토큰 기반 인증
- HTTPS 통신
- XSS/CSRF 방지
- 입력값 검증

## 테스팅 전략
- Playwright를 이용한 E2E 테스트
- 컴포넌트 단위 테스트
- API 통합 테스트

## 배포 및 운영
- 빌드 최적화
- 환경별 설정 관리
- 모니터링 및 로깅

1. 사용자 인증 시스템
2. 기본 UI/UX 구성
3. 반려동물 관리 기능
4. 일기 작성 및 조회 기능
5. 중고거래 마켓 기능
6. 실시간 채팅 기능
7. 관리자 기능
